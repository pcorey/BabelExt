Speed Reading Plugin & Bookmarklet

I'm working on a simple javascript based browser plugin. It's built on the BabelExt boilerplate which means it should run on Firefox, Chrome, Safari and Opera. Try it out on this page by selecting some text you wish to read and pressing <ALT> + r.
